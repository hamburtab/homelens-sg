# GeoJSON 数据清单

共 20 个文件，数据来源为新加坡 OpenStreetMap，总计约 18,000 个空间要素。

## 交通

- **BusStop.geojson** — 公交站（5,637 条）。全岛公交站点，含站名、经纬度。
- **geojson_railway_station.json** — 地铁站（247 条）。MRT/LRT 站点，含站名（中英马泰米尔文）、所属线路、运营商。

## 商超

- **geojson_mall.json** — 商场（258 条）。购物中心，含名称、地址、营业时间。
- **geojson_supermarket.json** — 超市（572 条）。各类超市。
- **geojson_convenience.json** — 便利店（707 条）。7-Eleven、Cheers 等。

## 吃喝

- **geojson_restaurant.json** — 餐厅（4,636 条）。全岛餐饮场所，含店名、地址、营业时间、电话。
- **geojson_cafe.json** — 咖啡厅（1,871 条）。咖啡馆及茶室。
- **geojson_food_court.json** — 食阁（782 条）。熟食中心、小贩中心。

## 休闲

- **geojson_park.json** — 公园（924 条）。邻里及区域公园。⚠️ 仅中心点，缺完整绿地边界。
- **geojson_nature_reserve.json** — 自然保护区（8 条）。武吉知马、中央集水区等。⚠️ 仅中心点。
- **geojson_sports_centre.json** — 体育中心（200 条）。体育馆、健身中心、泳池。

## 医疗教育

- **geojson_hospital.json** — 医院（70 条）。综合及专科医院，含名称、地址、电话。
- **geojson_clinic.json** — 诊所（479 条）。全科及专科诊所。
- **geojson_school.json** — 学校（552 条）。中小学、初级学院、国际学校。
- **geojson_kindergarten.json** — 幼儿园（247 条）。托幼机构。

## 公共设施

- **geojson_community_centre.json** — 社区中心（192 条）。民众联络所、社区俱乐部。
- **geojson_post_office.json** — 邮局（56 条）。SingPost 邮局网点。
- **geojson_atm.json** — ATM（1,088 条）。各银行自动取款机网点，含所属银行、地址。
- **geojson_bank.json** — 银行（307 条）。银行分行，含名称、地址、营业时间。
- **geojson_laundry.json** — 洗衣店（114 条）。干洗及自助洗衣店。
